Password Generator

The Password Generator is an app developed to create a random password for the user. The app first runs through a series of prompts regarding what the user would like their password to include (i.e. special characters, uppercase or lowercase letters, etc.) Once the user completes all prompts, a random password is generated for the user. This application was developed using basic HTML, CSS, and beginner Javascript.

Interactive Work Day Calendar

The Work Day Scheduler is an interative calendar app. The current date is displayed at the top of the calendar. The calendar allows the user to input data at the time they choose (each hour between 9AM - 5PM), and will save the input to local storage when the user presses the save button on the right side. When the user comes back to the app, their input will display where it has been entered prior. The calendar also displays different colors depending on the time of day; Gray meaning the time block has passed, red meaning the present time block, and green meaning future time block. This app was developed using HTML, CSS, Javascript, and Moment.js.

Project 1 - Image Gallery

For Project 1, my group and I developed an Image Gallery App. This app provides the user with a location to search for any type of image or GIF they would like. When the user inputs a keyword into the search bar and clicks the search icon, the image gallery will bring up images and GIFs related to the keyword. On the left side the user will find a dropdown with suggested Genres which will bring up images and GIFs releated to the genre when it is clicked. In addition, when an image or GIF is clicked on, a modal is dispayed on the screen where the user can "like" the image. This app was developed using Javascript, JQuery, ajax calls, and multiple API keys.
